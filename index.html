What Guide Is

Guide is built as an ongoing collaborator.

You can type to Guide like chat, or talk to him through your microphone. The difference is that Guide is built for continuity. He is meant to keep context, remember your preferences, track important facts, and stay useful over time.

That means the product is not just about answering one question. It is about helping with the next thing too, because Guide still knows what you were doing, what matters, and what should happen next.

How It Works

Simple on the surface. Layered where it counts.

01

Input

You type, talk, share context, or let Guide look at the screen.

02

Context

Guide combines your current request with recent conversation, saved memory, and optional live vision.

03

Decision

He decides whether the right move is answering, recalling memory, searching the web, or helping with a workflow.

04

Action

If action is allowed, Guide can navigate software, type, click, annotate the screen, or hand the next step back clearly.

Capabilities

What Guide can actually do.

Memory

Guide stores meaningful conversations, facts, preferences, knowledge, and verified details so context survives across sessions.

Vision

When vision is enabled, Guide can see your desktop, read interfaces, inspect layouts, and help with what is already on screen.

Voice

Guide can listen through your microphone and speak back, so working with him does not have to be keyboard-only.

Web Search

When current information matters, Guide can search instead of bluffing stale answers.

Computer Control

When explicitly enabled, Guide can click, type, scroll, open apps, and help handle multi-step desktop workflows.

Mobile Handoff

The mobile side is built to capture thoughts, reminders, observations, and context while you are away, then sync what matters back home.

Desktop Guide

The full Guide lives on your PC.

Desktop Guide owns the heavy stuff: deeper memory, richer tools, longer context, desktop automation, project workflows, and more serious reasoning.

Mobile Guide

The phone version is a companion, not a lie.

Mobile Guide is for capture, reminders, lightweight voice interaction, camera notes, and useful handoff. It is intentionally lighter, because pretending otherwise is bullshit.
